Information

Application Information
  • We currently support 3 exercises: Side Squats, Front Squats, and Side Push-Ups.
  • Unfortunately, as of now, it can only support one connection due to computing constraints.
  • The website supports both Desktop and Mobile access, in portrait and landscape mode.
  • The browsers we support Chrome, Safari, and Microsoft Edge. Firefox on phone cannot be used in landscape.
  • Do note that you will be logged out after 5 minutes of inactivity while using the application.
Data Privacy
  • Do note that by using the website, you are sending your image data to us.
  • We do not store or distribute this data in any way, shape, or form.
Disclaimer

We are not responsible for any injuries caused by using our application.
